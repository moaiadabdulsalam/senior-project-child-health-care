# senior-project-child-health-care

A pediatric health support platform connecting children, parents, and doctors through smart digital tools.

Built with [NestJS](https://nestjs.com/).

## Core Vision

To create a child-friendly digital health ecosystem that:

- Encourages proactive health monitoring
- Strengthens doctor-parent communication
- Engages children in a safe and interactive environment

## Project setup

```bash
npm install
```

## Compile and run the project

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Run tests

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
